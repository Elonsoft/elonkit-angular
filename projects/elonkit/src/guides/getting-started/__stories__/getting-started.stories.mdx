# Getting Started with Elonkit

Elonkit is the implementation of Elonsoft design system for Angular.

### Install

```
npm install @elonkit/angular
```

```
yarn add @elonkit/angular
```

After the packages have been successfully added to your project, you'll need to do the following:

1. [Setup palettes and typography](/?path=/docs/guides-theming--page);
2. [Setup icons](/?path=/docs/guides-icons--page);
3. [Setup locale](/?path=/docs/guides-localization--page).
