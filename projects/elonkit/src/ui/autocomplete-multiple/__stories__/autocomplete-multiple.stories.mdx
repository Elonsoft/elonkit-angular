## Constants

Injection token that can be used to configure the default options for all components within an app.

```ts
import { ES_AUTOCOMPLETE_MULTIPLE_DEFAULT_OPTIONS } from '@elonkit/angular/ui/autocomplete-multiple';

@NgModule({
  providers: [
    {
      provide: ES_AUTOCOMPLETE_MULTIPLE_DEFAULT_OPTIONS,
      useValue: {
        sizes: {
          chipLeftMargin: 4,
          countBadgeMaxWidth: 40
        }
      }
    }
  ]
})
```
